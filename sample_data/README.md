# Sample Data

This directory contains sample data files for testing Linquora 2.0.

## How to Use

Copy the files you need to the project root directory and rename them:

- sample_patients.txt     -> patients.txt
- sample_doctors.txt      -> doctors.txt
- sample_appointments.txt -> appointments.txt
- sample_rooms.txt        -> rooms.txt

Then run:

`ash
python hospital_management_system.py
`

The system will load this sample data automatically on startup.

## Sample Data Contents

- 3 patients (Ali Khan, Sara Ahmed, Bilal Raza)
- 4 doctors across different departments
- 3 appointments
- 4 rooms (General 101, ICU 102, Private 103, Semi-Private 104)
