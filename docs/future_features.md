# Future Features Wishlist

## Short Term (v2.1.0)

- Input validation for all fields (CNIC format, phone, age range)
- Date field for appointments (YYYY-MM-DD)
- Appointment conflict detection (same doctor, same time)
- Better ID generation (max + 1 instead of len + 1)
- Display patient name in appointment booking confirmation
- Room availability summary on main menu

## Medium Term (v2.2.0)

- CSV export for all records
- Import patients from CSV
- Appointment date range search
- Doctor schedule view (all appointments for a doctor by date)
- Basic statistics: total patients, doctors, occupied rooms
- Admission date tracking for patients

## Long Term (v3.0.0)

- Full GUI using tkinter
- SQLite database backend replacing flat files
- User authentication with role-based access (Admin, Doctor, Receptionist)
- Patient billing and invoice generation
- Report generation (daily patient list, appointment schedule)
- Multi-department support
- Discharge summary for patients
- Medicine prescription tracking
- Lab test records

## Community Suggested

- Dark mode CLI using colorama
- JSON export/import
- Email notifications for appointments
- Patient history timeline
