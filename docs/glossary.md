# Glossary

**Patient ID (PID)**
A unique auto-generated numeric identifier assigned to each patient at registration.

**Doctor ID (DID)**
A unique auto-generated numeric identifier assigned to each doctor.

**Appointment ID (AID)**
A unique auto-generated numeric identifier for each booked appointment.

**Room Number (RN)**
A user-defined string identifier for each room (e.g., "101", "A1").

**Room Type**
The category of a hospital room determining its capacity:
- General: 5 patients
- Semi-Private: 2 patients
- Private: 1 patient
- ICU: 1 patient

**Discharge**
The act of removing a patient from an assigned room (does not delete the patient record).

**Pipe-delimited**
A text file format where fields are separated by the | character.

**Bulk Operation**
An operation performed on multiple records at once using comma-separated IDs.
