# Linquora 1.0 vs Linquora 2.0

## Feature Comparison

| Feature | v1.0 | v2.0 |
|---------|------|------|
| Patient registration | Yes | Yes |
| Doctor management | Yes | Yes |
| Appointment booking | Basic | Yes |
| Room management | No | Yes |
| Room capacity enforcement | No | Yes |
| Bulk patient deletion | No | Yes |
| Bulk room discharge | No | Yes |
| Search patient by name | No | Yes |
| Search doctor by department | No | Yes |
| Appointment filter by patient | No | Yes |
| Appointment filter by doctor | No | Yes |
| Clear all appointments | No | Yes |
| Auto-room cleanup on delete | No | Yes |
| Docstrings on all functions | No | Yes |
| .gitignore | No | Yes |
| MIT License | No | Yes |
| Contributing guide | No | Yes |
| Changelog | No | Yes |
| Roadmap | No | Yes |

## Summary

Linquora 2.0 is a significant upgrade over 1.0 with full room management,
search capabilities, bulk operations, and comprehensive documentation.
